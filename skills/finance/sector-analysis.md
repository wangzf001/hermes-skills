## 分析流程

### Step 1：板块行情扫描
拉取板块涨幅 + 领涨个股 + 成交额

### Step 2：产业链地图
画出 上游→中游→下游 完整链条，标注：
- 各环节代表公司
- 国产化率/自给率
- 技术难点/进入壁垒

### Step 3：白毛魔女五维度卡口检测

| 维度 | 问题 | 是/否 |
|------|------|:--:|
| 物理稀缺 | 产能全球只有1-2家能做？ | |
| 技术独占 | 专利/工艺别人短期追不上？ | |
| 位置垄断 | 卡在产业必经之路？ | |
| 替代真空 | 下游不用它的产品就做不出？ | |
| 扩产壁垒 | 新建产能需要3年以上？ | |

> **≥3个"是" = 硬卡口 | 1-2个"是" = 软卡口 | 0个 = 伪卡口**

### Step 4：标的筛选
从产业链中筛选卡口环节的核心标的：
- 龙头：卡口环节的市场第一
- 弹性：小市值+高纯度
- 上游：产业链更上游的受益者

### Step 5：赔率对比
龙头已涨 vs 卡口未涨 → 如果逻辑兑现，谁的空间更大？

### Step 6：口播生成
调用 douyin-script-generator 的10条铁律，生成板块分析口播文案。
