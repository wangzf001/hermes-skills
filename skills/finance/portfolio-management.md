# 低频组合管理

为30万以下本金设计的集中持仓+因子驱动的低频交易框架。核心原则：**少而精、慢操作、留现金**。

## 组合铁律

1. **4-5只股票**，不再多。分散注意力比分散风险更致命
2. **集中2个方向**，不超过3个
3. **现金≥20%**，永远留弹药
4. **每周或双周一扫**，除非单日波动>10%才破例
5. **月交易≤10笔**

## 操作节奏

```
每月1号+15号 → factor_scoring.py 自动跑 → 推微信
看分行动：
  分数>65 → 持有
  45-65 → 观望  
  <45 → 减仓

止损（随时触发，不受日期限制）：
  -15% → 无条件砍
  +40% → 减1/3锁定利润
```

## 五因子打分

| 因子 | 权重 | 数据源 |
|------|:--:|------|
| 动量 | 25% | akshare stock_zh_a_hist |
| 价值 | 25% | PE+波动率 |
| 质量 | 20% | akshare stock_financial_abstract |
| 成长 | 20% | 归属母公司净利润增长率趋势 |
| 情绪 | 10% | 成交额变化 |

脚本：`scripts/factor_scoring.py`

## 建仓节奏

分3批，间隔2周：第1批50%（核心+防御）→ 第2批25%（弹性）→ 第3批25%（补仓）。跌了更好，同样钱买更多股。
